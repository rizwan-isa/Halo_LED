Voltage Measurement Board

%%PrID%% - Datapack - %%GitID%% - %%date%%

About

Verivolt module replacement PCB.

This board contains up to three independent high voltage measurement channels equivalent to three verivolt modules.

Review Process

Once the project is ready for review a Review Datapack should be generated and the commit should be tagged with the review name.

For example: PCB3_R2 > Round two of review of the third version of the PCB.

A branch with the same name should also be created so that review feedback can be recorded and changes be made to the project in accordance to the review notes.

Once all review issues have been addressed the branch should be merged into main and a new review datapack created for prototyping/production with one of the following tages PCB3_p (prototyping datapac) or PCB3_P (production datapack) this datapack should contain all the reviews raised during round 2 and the solutions provided. If the solutions provided are not satisfactory and another review stage is required at this stage a new with the PCB3_R3 name should be created and the PCB3_p tag should be renamed to PCB3_R3. Ideally solutions provided should be shared in advance with the reviewer so that the number of rounds and overhead is kept to a minimum.

Datapack Content

Review Data

All files in the datapack should be used for the review process.
The generated datapack contains all files required to perform an in-depth design review.

PCB Manufacturing Data

The review datpack contains all the data (gerber, etc.) required to produce blank PCBs these files are all zipped up and ready to be sent/uploaded onto a manufacturer like JLCPCB.

In order to limit unnecessary IP sharing only the 📜 %%PrID%%_%%PCBNr%%_%%GitID%%_%%date_only%%_PCB_gerbers.zip archive should be used for this purpose and no additional files should be shared!

PCBA Manufacturing Data

The review pack contains all required data for a PCBA manufacturer to produce and assembly PCBAs. This data is located in the 📂PCBA_%%PrID%%_%%PCBANr%%_%%GitID%%_%%date_only%% folder. Only files within this folder should be shared with the PCBA manufacturer in order to limit unnecessary IP sharing.
